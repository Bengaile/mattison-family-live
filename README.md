# Mattison Family Website

A private digital family museum preserving the stories, history, photos, recipes, pets,
and traditions of the Mattison family — built with [Astro](https://astro.build) and
deployed via GitHub Pages, same stack as momery.live.

## Setup

```bash
npm install
npm run dev
```

Visit http://localhost:4321

## Build

```bash
npm run build
```

Output goes to `/dist`.
