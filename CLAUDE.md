# Project Vision — MattisonFamily.org

This is a private family legacy website for the Mattison family. It preserves family
history, stories, photos, videos, pets, close friends, Mattison Farms, Clemson/Furman
connections, recipes, documents, and memories for future generations.

**Boundary:** This site is separate from MoMery / deer camp. No deer camp friends or
deer camp content belongs here.

**Guiding question for every page:** "Will someone 50 or 100 years from now better
understand who the Mattison family was because this page exists?" If yes, it belongs.

## Design Style
Warm, classic, elegant, personal, easy to navigate, readable for older family members —
a digital family museum, not social media.

## Color Palette
- Deep heritage green (primary)
- Warm Clemson orange (accent, used sparingly)
- Soft gold (secondary accent)
- Cream/linen background
- Warm charcoal text

## Tech Stack
Astro + Tailwind CSS, deployed to GitHub Pages with a custom domain (mattisonfamily.org).
File-based content using JSON in `src/data/`. No database required for v1.
